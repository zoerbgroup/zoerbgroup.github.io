Chemistry Acid-Base Titration Simulator

Interactive pH and fractional-composition plots for mono- through pentaprotic weak acids titrated with strong base.

Controls

Titration Curve Lookup
Lookup result
Enter a volume or pH to calculate a corresponding value on the titration curve.

Model assumptions: weak polyprotic acid titrated with strong base. pH is computed with a full charge-balance solver using polyprotic distribution fractions, which smooths the curve through all buffer and equivalence regions. Fractional composition is computed from the same alpha fractions.

Hover over either chart to see live crosshair coordinates. Click the pH plot to pin the nearest computed point.

Graphical Output

pH Titration Curve
Green curve = pH vs. added base volume. Gold marker = selected point. White crosshair = live hover. Purple dashed curve = first derivative overlay when enabled.
